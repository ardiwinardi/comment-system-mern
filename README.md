# Comment System with MERN Stack

This app demonstrates how to develop a comment system using MERN stack with monorepo

## Features

- Authorization
- Add, edit, delete comments
- Like, dislike comments
- Filter comments

## Installation

```bash
$ npm install
```

## Running app

```bash
# running all apps
$ npm run start

# running app
$ npm run start:app

# running api
$ npm run start:api
```
